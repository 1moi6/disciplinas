Introdução às Equações Diferenciais

Aula 1 — O que é uma equação diferencial e por que ela aparece

Uma pergunta simples — sem equações

Você prepara um café a 90 °C. A sala está a 20 °C.

Quanto tempo leva até o café atingir 60 °C?

O que você sabe:

  • O café começa quente e esfria.
  • Quanto maior a diferença de temperatura, mais rápido esfria.
  • Eventualmente chega à temperatura da sala.

O que você não sabe:

  • Em que instante exatamente?
  • A que taxa o resfriamento ocorre?
  • Como calcular isso?

Para responder com precisão, precisamos de uma linguagem melhor.

O que a intuição diz sobre a taxa de resfriamento?

Seja \(T(t)\) a temperatura do café no instante \(t\) (em minutos). A linguagem verbal já contém a ideia-chave:

“quanto maior a diferença, mais rápido esfria”

Em símbolos, uma afirmação sobre a taxa de variação de \(T\):

\[\frac{dT}{dt} \quad \text{depende de} \quad T(t) - T_m\]

onde \(T_m = 20\,°C\) é a temperatura do meio.

Pergunta

Qual é exatamente essa dependência?

Como transformar a intuição em equação?

Hipótese 1 — Temperatura uniforme

Enunciado

O café tem uma única temperatura em cada instante \(t\): a função \(T(t)\) está bem definida.

Custo: não modelamos gradientes internos nem a xícara em si.

Quando é razoável:

  • Xícara pequena, líquido bem misturado.
  • Não vale para um forno ou para o oceano.

Ganho

\(T\) é uma função de uma variável — \(t\).

Consequência matemática:

Podemos falar de \(\frac{dT}{dt}\) — a derivada de \(T\) em relação ao tempo está bem definida.

Hipótese 2 — Lei de Newton do Resfriamento

Enunciado A taxa de variação de \(T\) é proporcional à diferença \(T(t) - T_m\).

Custo: falha para grandes diferenças de temperatura ou se \(T_m\) variar ao longo do tempo.

\[\frac{dT}{dt} = k\bigl(T(t) - T_m\bigr), \qquad k < 0\]

A equação emergiu — não foi postulada

Cada símbolo carrega uma hipótese:

  • \(T(t)\): Hipótese 1 — temperatura bem definida.
  • \(k\): Hipótese 2 — proporcionalidade; \(k<0\) porque o café esfria.
  • \(T_m\): temperatura do meio, considerada constante.

Compare: dois tipos de equação

Equação algébrica

\[x^2 + 5x + 4 = 0\]

Incógnita: um número \(x\).

Solução: \(x = -1\) ou \(x = -4\).

Equação diferencial

\[\frac{dT}{dt} = k(T - T_m)\]

Incógnita: uma função \(T(t)\).

Solução: uma função que satisfaz a relação.

Conclusão 1

Uma equação diferencial (ED) é uma equação que envolve uma função desconhecida e suas derivadas. A incógnita é uma função, não um número.

Classificação: vocabulário para descrever EDOs

\[\frac{dT}{dt} = k(T - T_m)\]

Tipo

Derivadas em relação a uma variável (\(t\))? \(\Rightarrow\) EDO (ordinária)

Derivadas parciais? \(\Rightarrow\) EDP (parcial)

Ordem

Maior derivada presente: \(\frac{dT}{dt}\) (1ª derivada) \(\Rightarrow\) Ordem 1

Linearidade

\(T\) e \(T'\) aparecem ao 1º grau; coeficientes dependem de \(t\) \(\Rightarrow\) Linear

Mais exemplos de classificação

Regra para linearidade

A EDO de ordem \(n\) é linear se \(y,\, y',\, \ldots,\, y^{(n)}\) aparecem ao 1º grau e os coeficientes dependem apenas da variável independente.

Equação Ordem Linear?
\(\frac{dT}{dt} = k(T - T_m)\) 1 Sim
\(\frac{d^2y}{dx^2} - 2\frac{dy}{dx} + y = 0\) 2 Sim
\(\frac{dy}{dx} = xy^{1/2}\) 1 Não
\(\frac{d^2y}{dx^2} + 5\!\left(\frac{dy}{dx}\right)^{\!3} - 4y = e^x\) 2 Não

O que é uma solução?

De volta ao café: \(\frac{dT}{dt} = k(T - 20)\).

Afirmação: a função \(T(t) = 20 + 70\,e^{kt}\) satisfaz a equação.

\[\frac{dT}{dt} = 70k\,e^{kt}\] \[k(T-20) = 70k\,e^{kt} \quad \checkmark\]

Definição — Solução de uma EDO

\(\phi(t)\) definida em \(I\) é solução se, ao substituir na equação, resulta numa identidade em \(I\).

\(T(t) = 20 + Ce^{kt}\) é solução para qualquer \(C\) — solução geral.

Problema de Valor Inicial (PVI)

O problema

A família \(T(t) = 20 + Ce^{kt}\) tem infinitas soluções.

O café começa a 90 °C em \(t = 0\):

\[\frac{dT}{dt} = k(T - 20), \qquad \underbrace{T(0) = 90}_{\text{condição inicial}}\]

Aplicando a condição inicial

\[T(0) = 20 + C = 90 \;\Rightarrow\; C = 70\]

Solução particular

\[\boxed{T(t) = 20 + 70\,e^{kt}}\]

Definição — PVI

Um problema de valor inicial é uma EDO acompanhada de condições sobre a função em um único ponto \(t_0\). Resolver o PVI é encontrar a solução particular.

O que a equação já nos diz — sem resolver

Se \(T > 20\):

\(T - 20 > 0\) e \(k < 0\)

\[\Rightarrow \frac{dT}{dt} < 0\]

O café esfria. ✓

Se \(T < 20\):

\(T - 20 < 0\) e \(k < 0\)

\[\Rightarrow \frac{dT}{dt} > 0\]

Um objeto frio aquece. ✓

Equilíbrio

\(\frac{dT}{dt} = 0 \;\Leftrightarrow\; T = 20\). A solução constante \(T(t) = 20\) é a solução de equilíbrio.

Essa análise qualitativa — sem resolver — é uma das três abordagens do curso: analítica, qualitativa, numérica.

Uma equação, muitos fenômenos

\[\frac{dy}{dt} = ky\]

Crescimento populacional \(y = P(t)\) — população; \(k > 0\): crescimento. (Malthus, 1798)

Decaimento radioativo \(y = A(t)\) — massa; \(k < 0\): decaimento. (datação C-14)

Juros compostos \(y = S(t)\) — capital; \(k = r > 0\): taxa. (capitalização contínua)

Conclusão 2 Uma única EDO pode servir como modelo para fenômenos muito diferentes. O que muda é a interpretação dos símbolos.

Síntese — Como chegamos até aqui

Passos 1–3

  1. Partimos de um problema concreto: resfriamento do café.
  2. Formulamos duas hipóteses (temperatura uniforme; proporcionalidade).
  3. A EDO emergiu das hipóteses: \(\frac{dT}{dt} = k(T - T_m)\).

Passos 4–6

  1. Definimos solução como função que satisfaz a equação identicamente.
  2. O PVI seleciona uma solução particular da família.
  3. A análise qualitativa extraiu informação antes de resolver.

Onde estamos

Temos a solução \(T(t) = 20 + 70\,e^{kt}\), mas ainda não sabemos \(k\).

Para determiná-lo a partir de dados, precisamos resolver a equação de forma sistemática.

Uma observação útil

\[\frac{dT}{dt} = k(T - 20) \quad\Longrightarrow\quad \frac{dT}{T - 20} = k\,dt\]
